# Avatar Modular Assistant for Refinement & Integration
**An avatar modification assistant designed around a non-destructive workflow**

Avatar Modular Assistant is an Editor-based tool for Unity that assists with avatar refinement and outfit management while preserving a non-destructive workflow.  
It focuses on organizing, previewing, and preparing avatar modifications without directly altering original assets.

### Outfit Management

#### Group Management
- Create and manage outfit groups
- Organize multiple outfit elements under logical groupings
- Enable structured avatar customization workflows

#### Preview
- Preview individual outfit groups directly in the Unity Editor
- Ensure only one preview group is active during preview mode
- Designed for safe and reversible testing

#### Build Inclusion Control
- Toggle whether each outfit group is included in the final build
- Maintain a clear separation between preview state and build state
- Support flexible configuration before upload
